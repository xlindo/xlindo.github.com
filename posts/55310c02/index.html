树莓派-用树莓派对电机进行控制

xlindo

2015-03-04

备注：身体不适，工作进度有所放慢

前言

当前阶段的目标是，使用ssh远程操作，实现小车的运动控制。所以，第一步，就是使用树莓派对电机进行转动控制，也是运动控制的前期测试。

准备

树莓派
无线模块
L298N电机驱动板
移动电源
搭好的小车
电池组
线材若干

进行

  1. 将小车搭好，将电机线连至L298N电机驱动板，使用电池组对驱动板进行供电，将供电线正极连至12v输入，负极连至接地端并将其与树莓派板共地。
  2. 使用移动电源对树莓派单独供电，使用无线模块将其连入局域网，使用其他电脑对其远程控制。这里使用PuTTY的SSH远程登陆。
  3. 将L298N驱动板上的IN1~IN4使用杜邦线与树莓派GPIO口连接，这里使用GPIO口的0~3。

我们使用的马达连接驱动板后,IN1 IN2 是控制底盘左边的马达, IN3 IN4是控制底盘右边的马达. 控制的原理拿左边的电机来讲: 依次把驱动板的IN1~IN2置为高电平，就可以驱动马达，也就是说，要把树莓派的2个GPIO输出口依次置为高电平。例如，假设用0代表低电平，1代表高电平的话，GPIO 0、1口的电平第一次被置为1、0，第二次被置为0、1，持续下去,电机就会运作。

代码

  1. 将代码写入motorTest.c
  2. 下载安装wiringPi，该库用来进行io控制，源自Arduino：
    sudo apt-get install git-core
    git clone git://git.drogon.net/wiringPi
    cd wiringPi
    ./build
    还可以对库进行更新
    cd wiringPi
    git pull origin
  3. 编译motorTest.c:
    gcc motorTest.c -o motorTest -lwiringPi
  4. 运行：
    sudo ./motorTest 0 1 2 3

总结

  • 电机需要在较高电压下才能实现转动速度的提升
  • 使用wiringPi库对树莓派进行GPIO等的控制
  • 该方案是使用c语言完成的，而树莓派的主打语言是python，所以还需要进一步使用python对其进行编程控制，这将在后期完成

  2. python语言下的控制程序编写
