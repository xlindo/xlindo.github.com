0 ANSI标准库

标准库不是C语言本身的一部分，其函数、类型以及宏分别在下面的头文件中定义：

以下划线开头的外部标识符保留给标准库使用，同时，以下划线、两个下划线或大写字母开头的标识符也保留给标准库使用。

1 输入输出

该头文件中定义的输入和输出函数、类型以及宏的数目几乎占整个标准库的三分之一。

1.1 文件操作

  • FILE *fopen(const char *filename, const char *mode)：其中mode可以是"r"、"w"、"a"、"r+"、"w+"、"a+"之一。
    后三种方式允许对同一文件进行读写。在读写交叉过程中，必须调用fflush或文件定位函数。一次最多打开FOPEN_MAX个文件。
  • FILE *freopen(const char *filename, const char * mode, FILE *stream)：该函数以mode方式打开指定文件，并将其关联到stream指定的流。常用于改变与stdin、stdout和stderr相关联的文件。
  • int fflush(FILE *stream)：对输出流，将写入缓冲区但尚未保存的数据保存；对输入，结果未定义。如果发生错误返回EOF，否则返回0。fflush(NULL)将清洗所有输出流。
  • int fclose(FILE *stream)：该函数将所有未写入的数据写入stream，丢弃缓冲区中所有未读入数据，并释放自动分配的缓冲区，关闭流。如果发生错误返回EOF，否则返回0。
  • int remove(const char *filename)：删除指定文件，若失败，返回一个非0值。
  • int rename(const char *oldname, const char *newname)：重命名，若失败，返回一个非0值。
  • FILE *tmpfile(void)：以"wb+"创建一个临时文件，该文件在被关闭或程序正常结束时被删除。创建成功，返回一个流；创建失败返回NULL。
  • char *tmpnam(char s[L_tmpnam])：tmpnam(NULL)创建一个与现有文件名不同的字符串，并返回一个指向一内部数组的指针。tmpnam(S)函数把创建的字符串保存到数组S中，并将它作为函数值返回。该函数只是用于创建一个名字，而不是一个文件。
  • int setvbuf(FILE *stream, char *buf, int mode, size_t size)：在执行读写或其他操作前必须执行此函数，该函数控制流的缓冲。当mode为_IOFBF时，进行完全缓冲；当mode为_IOLBF时，对文本文件进行缓冲；当mode为_IONBF时，不缓冲。如果buf的值不是NULL，则将buf指向的区域作为缓冲区，否则将分配一个缓冲区。size决定缓冲区长度。如果出错，返回一个非0值。
  • void setbuf(FILE *stream, char *buf)：如果buf值为NULL，关闭流的缓冲；否则等价于(void)setvbuf(stream, buf, _IOFBF， BUFSIZE)。

1.2 格式化输出

  • int fprintf(FILE *stream, const char *format, ...)：按照说明的格式对输出进行转换，并写到流中。返回实际写入的字符，否则返回负值。格式串由普通字符和转换说明组成。转换说明以%开头，以转换字符结束。其间可以依次包括：

    • 标志，用于修改转换说明：
      • -：指定被转换的参数在其字段内左对齐；
      • +：指定输出的数前加上正负号；
      • 空格：如果第一个字符不是正负号，则加一个空格；
      • 0：当输出长度小于字段宽度，前加0填充；
      • #：指定另一种输出形式：如果为o转换，则第一个数字为0，如果为x或X转换，则指定在输出的非0值前加0x或0X；对于e、E、f、g、G转换，指定输出总包含一个小数点；对于g或G，保留末尾的0。
    • 一个数值，指定最小字段宽度
    • 点号，用于分隔字段宽度和精度
    • 精度：对于字符串，它指定打印字符的最大个数；对于e、E或f转换，指定小数点后的数字位数；对于g或G，指定有效数字位数；对于整型，指定打印的数字位数

    • 长度修饰h(short)、l(long)或L(long double)

      宽度和精度都可以用*指定，则该值将通过转换下一个参数（必须为int）计算得到。

  • int printf(const char *format, ...)：等价于fprintf(stdout, ...)
  • int sprintf(char *s, const char *format, ...)：与printf基本相同，但输出将被写到s中，并以\0结束
  • vprintf、vfprintf和vsprintf这三个函数和对应的printf等价，但用宏va_start初始化的arg代替了可变参数

1.3 格式化输入

  • int fscanf(FILE *stream, const char *format, ...)
  • int scanf(const char *format, ...)
  • int sscanf(const char *s, const char *format, ...)

1.4 字符输入输出

  • int fgetc(FILE *stream)
  • char *fgets(char *s, int n, FILE *stream)
  • int fputc(int c, FILE *stream)
  • int fputs(const char *s, FILE *stream)
  • int getc(FILE *stream)：等价于fgetc，不同的是，当getc定义为宏时，可能多次计算stream的值
  • int getchar(void)：等价于getc(stdin)
  • char *gets(char *s)
  • int putc(FILE *stream)：等价于fputc，不同的是，当putc定义为宏时，可能多次计算stream的值
  • int putchar(int c)：等价于putc(c, stdout)
  • int puts(const char *s)
  • int ungetc(int c, FILE *stream)：把c写回到流中

1.5 直接输入输出

  • size_t fread(void *ptr, size_t size, size_t nobj, FILE *stream)：从流中读入最多nobj个长度为size的对象，并保存到ptr指向的数组。返回读取的对象数目。必须通过feof和ferror获得结果执行状态。
  • size_t fwrite(const void *ptr, size_t size, size_t nobj, FILE *stream)：如果发生错误，返回值将小于nobj。

1.6 文件定位

  • int fseek(FILE *stream, long offset, int origin)：设置文件位置，后续操作从新位置开始。其中origin可以被设置为SEEK_SET（开始）、SEEK_CUR（当前）或SEEK_END（末尾）。对于文本流，offset必须设置为0或是ftell返回的值（此时origin必须是SEEK_SET)。该函数出错时返回一个非0值。
  • long ftell(FILE *stream)：返回stream当前位置，出错时返回-1L。
  • void rewind(FILE *stream)：等价于fseek(fp, 0L, SEEK_SET)、clearerr(fp)的执行结果。
  • int fgetpos(FILE *stream, fpos_t *ptr)：记录当前位置，供fsetpos使用。
  • int fsetpos(FILE *stream, const fpos_t *ptr)：设置位置。

1.7 错误处理

当发生错误或到达文件末尾时，标准库中许多函数都会设置状态指示符。另外，errno（<errno.h>）可以包含一个错误编号。

  • void clearerr(FILE *stream)：清除与流先关的文件结束符和错误指示符。
  • int feof(FILE *stream)：如果设置了与流相关的文件结束指示符，将返回一个非0值。
  • int ferror(FILE *stream)：如果设置了与流相关的错误指示符，将返回一个非0值。
  • void perror(const char *s)：将打印s以及与errno中整型值相应的错误信息，功能类似于fprintf(stderr, "%s: %s\n", s, "error message");。

2 字符类别测试

该头文件声明了一些测试字符的函数，若参数c满足指定条件，则返回非0值，否则反之。

  • isalnum(c)：isalpha或isdigit为真
  • isalpha(c)：isupper或islower为真
  • iscntrl(c)：为控制字符
  • isdigit(c)：为10进制数
  • isgraph(c)：为除空格外的可打印字符
  • islower(c)
  • isprint(c)：为包括空格的可打印字符
  • ispunct(c)：除空格、字母和数字外的可打印字符
  • isspace(c)：为空格、换页、换行、回车或制表符
  • isupper(c)
  • isxdigit(c)：为16进制数

3 字符串

该头文件定义了两组字符串函数，第一组以str开头命名，第二组以mem开头命名。下面的s和t为char *，cs和ct为const char *，n为size_t，c为int（将被转换为char）：

  • char *strcpy(s,ct)：复制ct到s。
  • char *strncpy(s,ct,n)：复制ct中n个字符到s，如果少于n个字符，用\0填充。
  • char *strcat(s,ct)：将ct连接到s的尾部。
  • char *strncat(s,ct,n)
  • int strcmp(cs,ct)：cs<ct时，返回负数；等于时，返回0；此外返回一个正数。
  • int strncmp(cs,ct,n)
  • char *strchr(cs,c)：返回指向c的第一次的位置指针，若不包含，则返回NULL。
  • char *strrchr(cs,c)：返回最后一次出现c的位置指针。
  • size_t strspn(cs,ct)：返回cs中包含ct的前缀的长度。
  • size_t strcspn(cs,ct)：与上面功能函数相反。
  • char *strpbrk(cs,ct)：返回一个位置指针，指向ct中任意字符第一次出现在字符串cs中的位置。
  • char *strstr(cs,ct)：指向ct第一次出现在cs中的位置。
  • size_t strlen(cs)
  • char *strerror(n)：返回指向错误编号对应的错误信息字符串。
  • char *strtok(s,ct)：在s中搜索由ct字符界定的记号。

以mem开头的函数按照字符数组的方式操作对象，主要目的是提供一个高效的函数接口。下面的s和t为void *，cs和ct为const void *，n为size_t，c为int（将被转换为unsigned char）：

  • void *memcpy(s,ct,n)
  • void *memmove(s,ct,n)
  • int memcmp(cs,ct,n)
  • void *memchr(cs,c,n)
  • void *memset(s,c,n)

4 数学

声明了一些函数和宏。
宏：

  • EDOM和ERANGE（<error.h>）：两个非0的整型常量，指示函数的定义域错误和值域错误。
  • HUGE_VAL：double类型的正数。

当参数位于函数定义的作用域外时，就会出现定义域错误。此时，全局变量errno的值将被设置为EDOM。如果函数的结果不能用double表示时，则会发生值域错误。结果上溢时，函数返回HUGE_VAL，并带有正确的正负号，errno的值被设置成ERANGE；下溢时，返回0，而errno是否设置要视具体情况。

下面的x和y类型为double，n的类型为int，返回值均为double：

  • sin(x)
  • cos(x)
  • tan(x)
  • asin(x)
  • acos(x)
  • atan(x)
  • atan2(y,x)
  • sinh(x)
  • cosh(x)
  • tanh(x)
  • exp(x)
  • log(x)：ln(x)
  • log10(x)
  • pow(x,y)
  • sqrt(x)
  • ceil(x)：不小于x的最小整数
  • floor(x)：不大于x的最大整数
  • fabs(x)
  • ldexp(x,n)：计算x.2n
  • frexp(x, int *ip)
  • modf(x, double *ip)：将数字分为正数和小数，符号一致正数保存在ip，小数返回。
  • fmod(x,y)：求浮点余数

5 实用函数

声明了一些执行数值转换、内存分配和其他类似工作的函数。

  • double atof(const char *s)：等价于strtod(s, (char**)NULL)。
  • int atoi(const char *s)：等价于(int)strtol(s, (char**)NULL, 10)。
  • long atol(const char *s)
  • double strtod(const char *s, char **endp)：将s的前缀转换成double类型，并在转换时跳过空白符。除endp为NULL，否则将把指向s中未转换部分的指针保存在*endp。
  • long strtol(const char *s, char **endp, int base)：将s的前缀转换成long。base值在2~36之间时，则以该数为基底。如果base为0，则基底为八进制、十进制或十六进制。
  • unsigned long strtoul(const char *s, char **endp, int base)：同上，为unsigned long类型。
  • int rand(void)：产生一个伪随机整数。
  • void srand(unsigned int seed)：将seed作为种子，其初值为1。
  • void *calloc(size_t nobj, size_t size)
  • void *malloc(size_t size)
  • void *realloc(void *p, size_t size)
  • void free(void *p)
  • void abort(void)：使程序非正常终止，类似raise(SIGABRT)。
  • void exit(int status)
  • int atexit(void (*fcn)(void))：登记函数，在正常终止时被调用。
  • int system(const char *s)
  • char *getenv(const char *name)：返回与name有关的环境的字符串。
  • void *bsearch(const void *key, const void *base, size_t n, size_t size, int (*cmp)(const void *keyval, const void *datum))：函数cmp比较两个参数。
  • void qsort(void *base, size_t n, size_t size, int (*cmp)(const void *, const void *))
  • int abs(int n)
  • long labs(long n)
  • div_t div(int num, int denom)：计算num/denom，并分别保存商和余数到div_t的成员quot和rem中。
  • ldiv_t ldiv(long num, long denom)

6 诊断

  • void assert(int expression)：执行时，若表达式为0，则该宏将在stderr中打印一条信息，如：Assertion failed: 表达式， file源文件， line行号，然后将调用abort终止程序。其中源文件名和行号来自于预处理器宏__FILE__和__LINE__。__如果定义了NDEBUG，则assert宏将被忽略。

7 可变参数表

该头文件提供了遍历未知数目和类型的函数参数表功能。假定函数f带有可变数目的实际参数，lastarg是它最后一个命名的形式参数。那么在函数内声明一个va_list的变量ap，它将依次指向每个实际参数：va_list ap;

在访问未命名的参数前，必须使用va_start宏初始化一次：va_start(va_list ap, lastarg);

此后，每次执行宏va_arg都将产生一个与下一个未命名的参数具有相同类型和数值的值，同时它还修改ap，以使得下一次执行va_arg时返回下一个参数：type va_arg(va_list ap, type);

在所有参数处理完毕之后，且在退出函数前，必须调用宏va_end一次：void va_end(va_list ap);

9 信号

该头文件提供了一些处理程序运行期间印发的各种异常条件的功能，比如来自外部的中断信号或程序执行错误引起的中断信号。

void (*signal(int sig, void (*handler)(int)))(int)：如果handler的值是SIG_DFL，则采用由实现定义的默认行为；若为SIG_IGN，则忽略；否则，调用handler指向的函数，有效的信号包括：

  • SIGABRT：异常终止
  • SIGFPE：算术运算出错，如除数为0或溢出
  • SIGILL：非常函数映像，如非法命令
  • SIGINT：用于交互式目的的信号，如中断
  • SIGSEGV：非法存储器访问，如访问不存在的内存单元
  • SIGTERM：发送给程序的终止请求

对于特定的信号，信号将返回handler的前一个值，如出现错误，返回SIG_ERR。

int raise(int sig)发送信号sig，若不成功，则返回一个非0值。

10 日期与时间

clock_t和time_t是两个表示时间的算术类型，struct tm用于保存日历时间的各个构成部分，包括：int tm_sec;、int tm_min;、int tm_hour;、int tm_mday;、int tm_mon;、int tm_year;、int tm_wday;、int tm_yday;、int tm_isdst;（夏令时标记）。

  • clock_t clock(void)：返回程序开始执行后占用的处理器时间，单位为秒。
  • time_t time(time_t *tp)：返回当前日历时间。
  • double difftime(time_t time2, time_t time1)：返回时间差，秒。
  • time_t mktime(struct tm *tp)：转换*tp的当地时间到日历时间。
  • char *asctime(const struct tm *tp)：格式化时间。
  • char *ctime(const time_t *tp)：转换日历时间到当地时间，等价于asctime(localtime(tp))。
  • struct tm *gmtime(const time_t *tp)：转换日历时间到UTC。
  • struct tm *localtime(const time_t *tp)：转换日历时间到当地时间。
  • size_t strftime(char *s, size_t smax, const char *fmt, const struct tm *tp)：格式化时间。
