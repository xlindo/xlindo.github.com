normrnd

Normal random numbers

r = normrnd(mu,sigma,sz1,...,szN) or r = normrnd(mu,sigma,[sz1,...,szN]) generates a sz1-by-⋯-by-szN array of normal random numbers.

normfit

Normal parameter estimates

  • [muHat,sigmaHat] = normfit(x) returns estimates of normal distribution parameters
  • [muHat,sigmaHat,muCI,sigmaCI] = normfit(x,alpha) specifies the confidence level for the confidence intervals to be 100(1–alpha)%.

2 PRBS, Pseudorandom binary sequence

idinput

Generate input signals. You can also use idinput to design an input that has sufficient bandwidth to excite the dynamic range of your system.

  • u = idinput(N,type,band,levels) in default, 1 channel (Nu=1), 1 period (M=1), band=[0, 1] (white noise), levels=[-1,1].
    • If set N=[P Nu M], nu is channel, P is period, M*P is the total length of the signal.
  • [u,freq] = idinput(___,'sine',Band,Range,SineData)
  • type
    • ‘rbs’ — Random binary signal
    • ‘rgs’ — Random Gaussian signal
    • ‘prbs’ — Pseudorandom binary signal
    • ‘sine’ — Sum-of-sinusoids signal
