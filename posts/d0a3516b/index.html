2. Gradient descent

  • $\theta_j := \theta_j - \alpha \frac{d}{d_\theta}J(\theta)$
    • work correctly
    • choose learning rate alpha
  • Multivariate linear regression
    • feature scaling: skewed elliptical shape -> circle
      • $-1 <= x_j <= 1$
      • mean normalization
        • $x_j = \frac{x_j - mu}{s}$ ; s:range
    • learning rate
    • polynomial regression
  • Normal equation
    • $\theta = pinv(X’X)X’*y$

3. Logistical Regression:

  • classification : $y=0\ or\ 1$; $0 <= h_\theta(x) <= 1$
  • sigmoid function: $h_\theta(x) =\theta^Tx$ -> $g(\theta^Tx) = g(z) = \frac{1}{1+e^{-z}}$
  • LinearRegressiong cost function:
    • $ J(\theta) = \frac{1}{m}\sum_{i=1}^{m}\frac{1}{2}(h_\theta(x^{(i)})- y^{(i)})^2)$
  • Cost Function:
    • $cost(h_\theta(x),y) = -y\log(h_\theta(x))-(1-y)\log(1-h_\theta(x))$
    • $J(\theta)=-\frac{1}{m}\sum_{i=1}^{m}[y^{(i)}\log(h_\theta(x))+(1-y^{(i)})\log(1-h_\theta(x))]$
  • Gradien Descent
    • $\theta_j := \theta_j - \alpha\frac{\partial}{\partial\theta}J(\theta)$
    • Repeat{ $\theta_j := \theta_j - \alpha\sum(h_\theta(x^{(i)})-y^{(i)})x^{(j)}_j$ }
    • Vectorized implementation
  • Process:
    • Given $\theta$, compute:
      • $J(\theta)$
      • $\frac{\partial}{\partial\theta_j}J(\theta)$
      • Regularization(except i=0)
        • $J(\theta)=\frac{1}{2m}[\sum_{i=1}^{m}(h_\theta(x^{(i)})-y^{(i)})^2+\lambda\sum_{j=1}^{n}\theta_j^2]$
        • $\theta_j := \theta_j - \alpha[\frac{1}{m}\sum_{i=1}^{m}(h_\theta(x^{(i)})-y^{(i)})x^{(i)}_j+\frac{\lambda}{m}\theta_j]$
  • Optimization algorithms
    • Gradient descent
    • Conjugate gradient
    • BFGS
    • L-BFGS
  • Pros
    • Not need to manually pick $\alpha$
    • Often faster than gradient descent
  • Cons
    • More complex
  • Octave
    • fminunc:function minimization unconstrained

4. Neural Networks

  • Notations
    • $a_i^{(j)}$: “activation” of unit i in layer j
    • $\theta^{(j)}$: matrix of weights controlling function mapping from layer j to layer j+1
      • e.g. $a_1^{(2)}=g(\theta_{10}^{(1)}x_0+\theta_{11}^{(1)}x_1+\theta_{12}^{(1)}x_2+\theta_{13}^{(1)}x_3)$
      • $h_\theta(x) = a_1^{(3)}$
      • $z^{(2)}=\Theta^{(1)}x=\Theta^{(1)}a^{(1)}$

5. NN BP

  • Logistic Regression: $J(\theta) = -\frac{1}{m}\sum_{i=1}^{m}[y^{(i)}\log((h_\theta(x^{(i)}))+(1-y^{(i)})\log(1-h_\theta(x^{(i)}))]+\frac{\lambda}{2m}\sum_{j=1}^{n}\theta^2_j$
  • Denote $h_\theta(x)_k$as $k^{th}$ -> NN: $J(\Theta)$
    • L: sum of layers; $s_l$:no. of units; k: no. of output units
    • Logistic Regression: $J(\Theta) = -\frac{1}{m}\sum_{i=1}^{m}\sum_{k=1}^{K}[y_k^{(i)}\log((h_\theta(x^{(i)})_k)+(1-y_k^{(i)})\log(1-(h_\theta(x^{(i)})_k))] + \frac{\lambda}{2m}\sum_{l=1}^{L-1}\sum_{i=1}^{s_l}\sum_{j=1}^{s_{l+1}}(\Theta_{j,i}^{(l)})^2$
  • Forward propagation
    • $a^{(1)}=x$
    • $z^{(2)}=\Theta^{(1)}a^{(1)}$
    • $a^{(2)}=g(z^{(2)})$; (add $a_0^{(2)}$)
    • $z^{(3)}=\Theta^{(2)}a^{(2)}$
    • $a^{(3)}=g(z^{(3)})$;(add $a_0^{(3)}$)
    • $z^{(4)}=\Theta^{(3)}a^{(3)}$
    • $a^{(4)}=g(z^{(4)})$
  • Backpropagration
    • intuition: $\delta_j^{(l)}$ = “error” of node j in layer l
    • PROCESS
        1. Given data set
        1. set$\Delta_{ij}^{(l)}:=0$ for all i,j
        1. for t=1 to m:
          • (1) set $a^{(1)}=x$
          • (2) FP-> compute $a^{(l)}$ for l=2,3,…,L
          • (3) compute $\delta^{(L)}=a^{(L)}-y^{(t)}$
          • (4) compute $\delta^{(L-1)},\delta^{(L-2)},\dots,\delta^{(2)}$, using $\delta^{(l)}=((\Theta^{(l)})^T\delta^{(l+1)}.a^{(l)}.(1-a^{(l)})$
          • (5) $\Delta_{ij}^{(l)}:=\Delta_{ij}^{(l)} +a_{j}^{(l)}\delta_{i}^{(l)}$
          • e.g. $\delta^{(3)}=(\Theta^{(3)})^T\delta^{(4)}.*g^{‘}(z^{(3)})$
          • e.g $g^{‘}(z^{(3)}) = a^{(3)}.*(1-a^{(3)})$
          • $\delta^{(4)}\rightarrow\delta^{(3)}\rightarrow\delta^{(2)}\rightarrow\delta^{(1)}$
        1. if $j\neq0$, $D_{i,j}^{(l)}:=\frac{1}{m}(\Delta_{ij}^{(l)} + \lambda\Theta_{ij}^{(l)})$; if $j=0$, $D_{i,j}^{(l)}:=\frac{1}{m}\Delta_{ij}^{(l)}$
        1. $\frac{\partial}{\partial\Theta_{i,j}^{(l)}}J(\Theta)=D_{i,j}^{(l)}$
  • Gradient checking
    • $\frac{\partial}{\partial\theta}J(\theta)\approx\frac{J(\theta+\epsilon)-J(\theta-\epsilon)}{2\epsilon}; \epsilon=10^{-4}$
    • delete these checking code when implement
  • Train a NN
      1. randomly initialize weights, cannot be set as 0
      1. FP to get $h_\theta(x^{(i)})$
      1. compute $J(\Theta)$
      1. compute $\frac{\partial}{\partial\Theta_{i,j}^{(l)}}J(\Theta)$
      1. use gradient check to compare BP vs. numerical estimate

6. Evaluation

  • how to improve?
    • fix high variance:
      • get more training examples
      • try smaller sets of features
      • try increasing $\lambda$
    • fix high bias:
      • try getting additional features
      • try getting polynominal features
      • try decreasing $\lambda$
  • evaluate
    • split, train:test = 7:3
    • split, train:cv:test = 6:2:2
    • compute error
    • learning curve
    • precision/recall
    • F1-score

7. SVM

  • $\min \limits_{\theta} C\sum_{i=1}^{m}[y^{(i)}cost_1(\theta^Tf^{(i)})+(1-y^{(i)})cost_0(\theta^Tf^{(i)})] + \frac{1}{2}\sum_{i=1}^{n}\theta_j^2$
  • $\rightarrow A+\lambda B$
  • hypothesis: y=1 if $\theta^Tf \geq 0$
  • f: kernel
    • e.g. Gauss: $f=e^{(-\frac{|x_1-x_2|^2}{2\sigma^2})}$
  • SVM vs. Logistic Regression
    • n: features; m: training examples
    • n >> m, LR or SVM without kernel
    • n << m, add features
    • n is small, m intermediate, SVM with kernel
